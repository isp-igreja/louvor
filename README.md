# Letras de Louvor — Como Editar pelo GitHub

Este repositório guarda as letras das músicas exibidas no site. Qualquer pessoa com acesso pode editar, renomear ou adicionar músicas **diretamente pelo site do GitHub**, sem precisar instalar nada no computador.

---

## Antes de começar

Você precisará de uma conta no GitHub. Se ainda não tiver, crie uma em [github.com](https://github.com) (é gratuito). Depois, peça para o responsável pelo repositório te adicionar como colaborador.

---

## Como editar a letra de uma música

1. Acesse o repositório no GitHub: **github.com/isp-igreja/louvor**
2. Clique na pasta **`musicas`**
3. Escolha a subpasta: **`Louvores`** ou **`Outros`**
4. Clique no arquivo da música que deseja editar (ex: `Salmo 19.txt`)
5. Clique no **ícone de lápis** (✏️) no canto superior direito da tela do arquivo
6. Faça as alterações diretamente no texto
7. Ao terminar, desça até a seção **"Commit changes"**
8. Escreva uma descrição breve do que foi alterado (ex: `Corrigiu letra do Salmo 19`)
9. Clique em **"Commit changes"**

O site será atualizado automaticamente em alguns minutos.

---

## Como renomear uma música (mudar o título)

O título exibido no site vem direto do **nome do arquivo**. Para renomear:

1. Abra o arquivo da música (siga os passos 1 a 4 acima)
2. Clique no **ícone de lápis** (✏️) para entrar no modo de edição
3. No topo da tela, clique sobre o **nome do arquivo** (o campo com o nome terminado em `.txt`) — ele vai ficar editável
4. Apague o nome antigo e digite o novo nome. **Mantenha sempre o `.txt` no final** (ex: `Novo título da música.txt`)
5. Desça até **"Commit changes"**, escreva uma descrição e clique em **"Commit changes"**

> **Atenção:** o arquivo existe em duas pastas — `Louvores` e `Outros`. Se quiser renomear nos dois lugares, repita o processo em cada pasta.

---

## Como adicionar uma música nova

1. Acesse o repositório: **github.com/isp-igreja/louvor**
2. Entre na pasta **`musicas`** e depois na subpasta desejada (`Louvores` ou `Outros`)
3. Clique em **"Add file"** (botão no canto superior direito) e depois em **"Create new file"**
4. No campo do nome do arquivo (topo da tela), digite o nome da música seguido de `.txt`
   - Exemplo: `Quão grande és Tu.txt`
5. Na área de texto abaixo, cole ou digite a letra completa da música
6. Desça até **"Commit changes"**, escreva uma descrição (ex: `Adicionou Quão grande és Tu`) e clique em **"Commit changes"**

---

## Dicas importantes

- **Nome do arquivo = título da música no site.** Use o nome exatamente como quer que apareça.
- **Não use o caractere `/` no nome do arquivo** — o GitHub interpretaria isso como uma subpasta.
- A pasta **`Louvores`** e a pasta **`Outros`** funcionam como categorias separadas no site. Salve o arquivo na pasta correta.
- Após um commit, aguarde alguns minutos para o site refletir a mudança.
- Se cometer um erro, é possível desfazer — fale com o responsável pelo repositório para reverter.
